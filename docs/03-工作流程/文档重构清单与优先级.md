# 【行动指南】文档重构清单与优先级

> **目的**：基于"弱信号组合哲学"，系统性重构现有文档  
> **核心变化**：从"追求强信号"转向"接受弱信号 + 组合"

---

## 一、核心认知变化总结

### 旧认知 → 新认知

| 维度 | 旧认知（错误） | 新认知（正确） |
|------|---------------|---------------|
| **信号强度** | 追求 PASS（p < 0.01） | 接受 WEAK（p < 0.20） |
| **验证标准** | 严格（方向+波动+KL 都要显著） | 灵活（任一显著即可用） |
| **判决结果** | PASS / FAIL | TIER 1 / TIER 2 / TIER 3 / FAIL |
| **策略数量** | 1-2 个强策略 | 5-10 个弱信号组合 |
| **系统目标** | 单策略高胜率（> 60%） | 组合后稳定性（夏普 > 1.5） |
| **核心能力** | 策略能力 | 风控能力 + 组合能力 |

问题 1：是否存在统计偏移？

问题 2：是否有经济学逻辑？
- 可解释的市场行为 → YES

- 4 个 YES → TIER 1（核心证人）

- < 2 个 YES → FAIL

## 五、关键原则（重构时必须遵守）

### 1. 不要推翻架构

```
✅ 保留：三权分立、证人制度、内核级风控
❌ 不要：重新设计架构

理由：架构设计已经完美支持弱信号组合
```

### 2. 只调整验证标准

```
✅ 修改：验证标准（PASS → TIER）
✅ 修改：判决逻辑（严格 → 灵活）
❌ 不要：修改风控逻辑
❌ 不要：修改执行逻辑

理由：只有验证标准需要调整
```

### 3. 强调组合而非单一

```
✅ 强调：多个弱信号组合
✅ 强调：证人投票机制
✅ 强调：动态权重调整
❌ 不要：追求单一强策略

理由：组合是系统的核心价值
```

### 4. 保持务实和保守

```
✅ 目标：年化 20-50%
✅ 目标：夏普 > 1.5
✅ 目标：不爆仓
❌ 不要：追求暴利
❌ 不要：过度自信

理由：长期生存 > 短期暴利
```
