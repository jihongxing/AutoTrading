# Settings 与 Admin 页面需求

## 概述
实现用户设置页面和管理员后台页面，包括 API Key 配置和移动端优化。

## 功能需求

### FR-1: Settings 用户设置
- FR-1.1: 账户信息展示（邮箱、订阅、注册时间）
- FR-1.2: 修改密码功能
- FR-1.3: 交易所 API Key 配置
- FR-1.4: API Key 验证功能
- FR-1.5: 通知设置开关

### FR-2: API Key 管理
- FR-2.1: 添加 API Key（加密传输）
- FR-2.2: 验证 API Key 有效性
- FR-2.3: 显示 API Key 状态（已验证/未验证）
- FR-2.4: 删除 API Key（二次确认）
- FR-2.5: API Key 仅显示后 4 位

### FR-3: Admin 管理后台
- FR-3.1: 平台统计概览（用户数、交易量、收益）
- FR-3.2: 用户列表（搜索、筛选、分页）
- FR-3.3: 用户详情查看
- FR-3.4: 用户状态管理（暂停/激活）
- FR-3.5: 系统操作（强制锁定）

### FR-4: 移动端优化
- FR-4.1: 底部 Tab 导航完善
- FR-4.2: 触摸手势支持
- FR-4.3: 下拉刷新
- FR-4.4: 骨架屏加载

## 非功能需求

### NFR-1: 安全
- 修改 API Key 需要密码确认
- 删除操作需要二次确认
- Admin 页面仅管理员可访问

### NFR-2: 移动端
- Settings 页面完整适配
- Admin 页面仅桌面端
