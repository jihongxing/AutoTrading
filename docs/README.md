## 🚀 快速开始（5 分钟了解系统）

### 系统是什么？

一个基于**弱信号组合**的 BTC 自动交易系统，通过：
- 🔍 **多个弱信号证人**（胜率 52-55%）组合
- 🛡️ **内核级风控**（保证系统生存）
- 🔄 **自学习优化**（持续进化）
- 📊 **大样本累积**（长期稳定盈利）

### 核心认知

> **在高效市场中，不存在"强信号"（胜率 > 60%）。**
> 
> **系统价值在于：弱信号组合 + 风控 + 大样本 = 长期盈利。**

### 系统目标

- 年化收益：20-50%（而非 > 100%）
- 最大回撤：< 20%
- 夏普比率：> 1.5
- 核心目标：**长期生存 > 短期暴利**

## 💡 关键原则（必须遵守）

### 1. 接受弱信号

```
❌ 追求：PASS（p < 0.01, 胜率 > 60%）
✅ 接受：WEAK（p < 0.20, 胜率 52-55%）

理由：高效市场中不存在强信号
```

### 2. 风控优先

```
风控 > 策略

即使策略失效，只要风控有效，系统依然可以生存
```

### 3. 组合优于单一

```
单个弱信号：52% 胜率
3 个组合：55-58% 胜率
5 个组合：58-62% 胜率
```

### 4. 长期优于短期

```
目标：年化 20-50%，夏普 > 1.5，生存 > 5 年
而非：月收益 100%，1 年暴利
```
